Empty state

An empty state appears when there is no data to display and describes what the user can do next.

The only required part of an empty state is the header.

Image dimensions

Setting the imageWidth and imageHeight props can be useful to prevent your layout from reflowing when images load in. If you have used the imageUrl approach, the imageWidth and imageHeight props set the width and height attributes on the underlying img element. If you have used the renderImage prop, they will be passed in as props to the render function. If you are resizing a spot illustration to try and squeeze it into a layout, it's a good time to ask whether you should include the image at all
